Nossa História

Fundado em 2013, o projeto JIU-JITSU - Disciplina e Educação para a Vida nasceu do sonho de utilizar a arte suave como ferramenta de mudança social. Oferecemos mais que técnicas de luta; oferecemos um caminho de cidadania e esperança para os jovens de comunidades carentes.

Perguntas Frequentes

Preciso ter experiência para começar?

Não! Nossas aulas são adaptadas para todos os níveis, desde o iniciante que nunca pisou no tatame até graduados.

Qual idade mínima para participar?

Atendemos crianças a partir de 7 anos na turma infantil. Para adultos, não há limite de idade.

Não tenho quimono, posso treinar?

Para as aulas experimentais, você pode vir com uma roupa esportiva confortável. Mas também possuímos alguns kimonos de uso geral disponíveis para você. Orientaremos sobre uma possível compra de um quimono depois.

O projeto é realmente gratuito?

Sim, o projeto social foca na educação e disciplina. Entre em contato para entender os critérios.
